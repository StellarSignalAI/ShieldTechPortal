Install the app

Tap the button below to install ShieldTech Tech on this device.

On iPhone & iPad, add ShieldTech to your Home Screen from Safari:

  1. Tap the Share icon at the bottom of Safari.
  2. Scroll down and tap Add to Home Screen.
  3. Tap Add — the ShieldTech icon appears on your Home Screen.

On Windows & Mac, install from Chrome or Edge:

  1. Click the Install icon in the address bar (right side), or open the browser menu.
  2. Choose Install ShieldTech Tech… / Apps → Install this site as an app.
  3. Confirm — it opens in its own window and lands in your Start Menu / Dock.

In Chrome on Android, install from the menu:

  1. Tap the ⋮ menu (top-right).
  2. Tap Install app (or Add to Home screen).
  3. Confirm — ShieldTech appears in your app drawer.
Or open the web app now

It's the same secure app whether you install it or open it in your browser — sign in with the credentials from your invite. Installing just adds a home-screen icon and full-screen mode.
